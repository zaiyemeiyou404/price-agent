# Price Agent 使用教程

这是一份最简单的使用教程。照着做即可。

## 1. 下载项目

打开命令行，进入你想放项目的文件夹，然后执行：

```bash
git clone https://github.com/zaiyemeiyou404/price-agent.git
```

进入项目：

```bash
cd price-agent
```

## 2. 安装 Python

如果电脑还没有 Python，先安装 Python。

建议安装 Python 3.10 或更高版本。

安装后检查：

```bash
python --version
```

能看到版本号就可以继续。

## 3. 创建虚拟环境

在项目目录里执行：

```bash
python -m venv .venv
```

启用虚拟环境：

```bash
.venv\Scripts\activate
```

看到命令行前面出现 `(.venv)` 就说明成功了。

## 4. 安装依赖

执行：

```bash
pip install -r requirements.txt
```

## 5. 安装浏览器组件

这个项目用 Playwright 爬取网页，需要安装 Chromium：

```bash
playwright install chromium
```

## 6. 创建配置文件

复制配置模板：

```bash
copy .env.example .env
```

一般情况下不用改 `.env`，先保持默认即可。

## 7. 启动后端服务

执行：

```bash
start.bat
```

如果不想用 `start.bat`，也可以执行：

```bash
python -m uvicorn app.main:app --host 127.0.0.1 --port 8000
```

启动成功后，不要关闭这个命令行窗口。

## 8. 打开接口页面

在浏览器打开：

```text
http://localhost:8000/docs
```

如果能看到 API 页面，说明后端启动成功。

## 9. 打开前端页面

双击项目里的：

```text
index.html
```

然后在页面里输入商品名称进行比价。

## 10. 测试接口是否正常

打开另一个命令行窗口，执行：

```bash
curl http://localhost:8000/health
```

如果返回：

```json
{"status":"ok"}
```

说明服务正常。

## 11. 常用命令

启动服务：

```bash
start.bat
```

查看接口文档：

```text
http://localhost:8000/docs
```

调试淘宝爬虫：

```bash
python scripts/debug/debug_crawler.py taobao "iPhone 15"
```

淘宝扫码登录：

```bash
python scripts/login/login.py taobao qr
```

京东扫码登录：

```bash
python scripts/login/login.py jd qr
```

## 12. 不要上传这些文件

这些文件和文件夹不要上传到 GitHub：

- `.env`
- `cookies/`
- `debug_output/`
- `__pycache__/`
- 截图文件
- 调试生成的 HTML 文件

项目里的 `.gitignore` 已经帮你忽略了它们。

## 13. 如果运行失败

先检查这几件事：

1. 是否已经启用虚拟环境。
2. 是否执行过 `pip install -r requirements.txt`。
3. 是否执行过 `playwright install chromium`。
4. 后端服务窗口是否还开着。
5. 浏览器打开的是不是 `http://localhost:8000/docs`。

如果还是不行，把报错内容复制出来再排查。
