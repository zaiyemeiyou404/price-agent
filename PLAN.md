# Price Agent 规划（纯爬虫版）

## 目标

- 基于 Playwright 抓取电商/外卖平台公开页面数据。
- 统一标准化商品结构，输出比价结果与跳转链接。
- 前端通过 `capabilities` 动态渲染平台，支持后续增量扩展。

## 当前约束

- 不接入任何第三方商品 API。
- 不依赖淘宝客/京东联盟/折淘客等密钥。
- `api_mode` 固定为 `false`，系统仅走爬虫路径。

## 分阶段计划

### Phase 1: 稳定淘宝单平台
- 强化登录态检测与 Cookie 续期流程。
- 搜索失败自动降级有头模式并落盘截图。
- 统一错误码和 summary 提示，减少“空结果无解释”。

### Phase 2: 多平台爬虫接入
- 恢复并验证 `jd/pdd/eleme/meituan` 的选择器链路。
- 平台级熔断（连续失败自动降级、短期跳过）。
- 结果聚合排序与平台分桶展示优化。

### Phase 3: 可观测性与回归
- 增加关键日志字段：query/platform/stage/error_code。
- 增加联调回归集（Postman/Newman）。
- 增加最小化健康检查脚本（服务、Redis、浏览器依赖）。

### Phase 4: 运行与运维
- 爬虫频控与重试策略参数化。
- Docker 化部署与进程守护。
- 价格监控任务（定时执行 + 消息通知）。

## 成功标准

- 单请求比价成功率稳定 > 90%（淘宝单平台）。
- 前端与接口字段一致，无硬编码平台列表。
- 服务可一键启动，`.env` 不含任何第三方商品 API 配置也可运行。
